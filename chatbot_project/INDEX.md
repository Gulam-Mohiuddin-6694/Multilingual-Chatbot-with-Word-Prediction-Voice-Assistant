## 🎯 Common Tasks

### Installation
```bash
# Quick (Windows)
start.bat

# Manual
pip install -r requirements.txt
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords'); nltk.download('wordnet')"
```

### Running
```bash
cd backend
python app.py
# Open: http://localhost:5000
```

### Testing
```bash
python test_system.py
python demo.py
python verify_installation.py
```
