## 🚀 How to Run

### Quick Start
```bash
cd chatbot_project
pip install -r requirements.txt
python -c "import nltk; nltk.download('punkt'); nltk.download('stopwords'); nltk.download('wordnet')"
cd backend
python app.py
```

### Access
Open browser: `http://localhost:5000`
